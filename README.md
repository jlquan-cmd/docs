# Docs2work Documentation

This folder contains the Mintlify documentation source for docs.docs2work.com.

## Setup

1. Push this folder to a separate GitHub repo (e.g. docs2work/docs)
2. Sign up at mintlify.com and connect the GitHub repo
3. Set the custom domain to docs.docs2work.com in Mintlify settings
4. The site will auto-deploy on every push to main

## Structure

- mint.json — Mintlify config, navigation, colors, branding
- introduction.mdx — Landing page
- authentication.mdx — Auth and key scopes
- quickstart.mdx — Nine-step build protocol
- concepts/ — Workflows, Forms, Portals, Audit Trail
- api-reference/ — One page per endpoint group (add these next)
- agents/ — Agent-specific guides
